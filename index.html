MY PROJECTS

Some projects that I’ve built over the past 8 years

Listing Alert

A Real Estate Communication tool meant to help brokerages create listing matches between sellers and buyers! The tech stack consisted of a mobile app built with React Native, an Admin Dashboard built with React.js, an API built with GraphQL and Node/Express, and a MongoDB database. The app served over 1800+ active users on a daily basis

Quark: Smart Home Mobile App

The Quark App, was the 2018 grand prize winner of the 59DaysOfCode startup competition in the Smart Cities category! The app was designed to help users monitor their home's daily energy expenditure, giving users the ability to set budgets with push notifications helping them conserve energy on a daily and monthly basis! The app consisted of a Mobile App built with Ionic, a web app app built with Angular, and a Node/Express/MongoDB backend.

No Place Like Home - Virtual Festival

No Place Like Home was the first of its kind Virtual Music Festival meant to allow users the ability to enjoy a music festival from the comfort of their homes during the pandemic lockdown. The festival attracted over 1000+ attendees and featured over 30+ performances, 20+ speakers and over 50 different vendors! The app was built with a React frontend tapping into the Zoom SDK to bring the events to users homes!

Holiday Magic - Web Application

Holiday Magic is a unique user experience that redesigns how families take their holiday photos! The app allows users to schedule their photo session at their convenience, uploads their photos to a web app in real time, then creates custom packages for families to choose from. The front end is built using Angular and utilizes the efficiency of AWS S3 buckets for photo storage, while the backend is built using Node/Express and PostgresQL. The app generates over $100,000 in gross revenue on a yearly basis!
